Parche 2.3b

Corrige Reparar ciudad:
- Si un POI no tiene POI_MASTER.md, ahora lo crea.
- No copia contenido de Catedral ni de otro POI.
- El archivo se crea limpio usando el nombre de la carpeta del POI.

Instalación:
1. Copiar el contenido del ZIP sobre Editor_UBIGUIA.
2. Reemplazar archivos.
3. Ejecutar python src/main.py.
4. Presionar Reparar ciudad.

# PROMPT_INVESTIGACION_v1

Versión: 1 — infraestructura únicamente. No es el Prompt Maestro definitivo.

## Objetivo

Investigar un POI (punto de interés) del Motor de Investigación de UBIGUIA
y producir un borrador de ficha (`POI_MASTER_BORRADOR.md`) junto con sus
fuentes, siguiendo la filosofía editorial del proyecto: no inventar
datos, conservar la trazabilidad de cada afirmación importante a una
fuente real, y dejar el contenido listo para revisión humana antes de
aprobarse.

## Alcance

Investigación de un único POI ya existente en Editor_UBIGUIA, a partir
de su nombre y de su contexto geográfico (ciudad, provincia, país,
categoría).

## Formato esperado

Un borrador de ficha en Markdown, compatible con la estructura de
`POI_MASTER_BORRADOR.md` ya definida por el Motor de Investigación:
identificación, descripción general, historia, qué observar,
curiosidades, información útil, resumen para audio, POIs relacionados,
palabras clave y fuentes.

## Estado de este prompt

**Este prompt todavía no está desarrollado.** Esta versión (v1) es
únicamente un marcador de infraestructura, para que
`ProveedorInvestigacionAnthropic` pueda cargar un archivo real desde
disco en lugar de tener el texto escrito dentro del código Python. El
contenido definitivo del Prompt Maestro de Investigación —con las
reglas editoriales completas, el formato exacto de respuesta y el uso
de búsqueda web— se escribirá en una etapa posterior.
